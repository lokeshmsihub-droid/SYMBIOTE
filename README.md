# SYMBIOTE | Behavior Intelligence & Gamification Showcase

SYMBIOTE is a high-fidelity frontend showcase of a SaaS behavior intelligence and gamification platform. It is built with a premium "AdaptoAI" Dark Mode aesthetic, focusing on quantifying user activity, maintaining consistency through streaks, and providing deep analytical insights.

> [!NOTE]
> **This is a Frontend-Only Demo**: All backend services have been decoupled and replaced with a high-fidelity mock data layer in `api.js`. This allows the platform to be browsed instantly without any database or server setup.

---

## 🛠️ Technology Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4 (Custom Emerald/Violet Design System)
- **Visuals**: Recharts (High-fidelity data visualization), Lucide React (Iconography)
- **Animations**: Framer Motion (Smooth transitions and micro-interactions)
- **Data Layer**: Native Mock API (Promises-based local data engine)

---

## 🎨 UI & Design Language

The platform uses a **Premium Dark Mode** design system called **AdaptoAI**.

### **Key UI Features**
- **The "Hat" Header**: A high-impact currency bar in the Rewards section that anchors user motivation.
- **Protection Arsenal**: High-density horizontal cards in the Streak system that manage "Streak Freeze" and "Shields".
- **Smart Analytics**: 4-metric grid synchronization with smooth Area and Bar charts for performance tracking.
- **Dynamic Leaderboard**: Global ranking system with multi-dimensional filtering (Quality vs. Speed).

---

## ⚙️ Core Modules Breakdown

### **1. Streak Management Showcase**
Tracking daily consistency with built-in protection mechanisms.
- **Consistency Map**: 30-day behavioral heatmap.
- **Streak Flame**: Tiered visual feedback based on intensity.

### **2. Intelligence Analytics**
Real-time quantification of "Evolutionary Behavior Patterns."
- **Trend Charts**: Tracking XP and Score correlations.
- **Categorical Breakdown**: Visualizing balance across Engineering, Leadership, and Innovation.

### **3. Rewards Store**
A gamified marketplace for redeeming behavioral performance points for high-value vouchers and perks.

---

## 🚀 Getting Started

To run the platform locally in development mode:

```bash
cd client
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.
